# schooltest-web

Next.js 16 frontend for the Schooltest platform.

## Stack

- Next.js 16 (App Router, React 19, Turbopack)
- TypeScript 5
- Tailwind CSS v4
- shadcn/ui
- next-intl
- TanStack Query
- Zustand
- Playwright (E2E)

## Development

```bash
pnpm install
pnpm dev
```

## Checks

```bash
pnpm typecheck
pnpm lint
pnpm test:e2e
```
